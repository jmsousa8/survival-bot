# Soulmask Bot

Discord webhook bot for Soulmask dedicated server. Monitors server log files and sends notifications for game events to Discord.

## Features

- Player death notifications
- Server start/stop notifications

## Getting Started

### Configuration

Create a `.env` file in the project root:

```bash
# Copy the example config
cp .env.example .env
```

Edit `.env` with your values:

| Variable            | Required | Description                                                |
|---------------------|----------|------------------------------------------------------------|
| `LOG_FILE_PATH`     | Yes      | Path to WS.log (e.g., `C:\Users\...\WS\Saved\Logs\WS.log`) |
| `DISCORD_WEBHOOK`   | No       | Discord webhook URL (leave empty for dry-run)              |
| `GEMINI_API_KEY`    | No       | Google Gemini API key (free tier available)                |

#### Getting a Discord Webhook

1. Open your Discord server
2. Go to Server Settings → Integrations → Webhooks
3. Create New Webhook
4. Copy the URL

#### Getting a Gemini API Key

1. Go to [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Create an API key (free tier: 15 requests/min, 1M tokens/month)
3. Copy the key to `GEMINI_API_KEY` in `.env`
4. Set `LLM_PROVIDER=gemini`

**Note:** When LLM is enabled, death messages will be generated by AI. If the LLM fails, it falls back to the default static message.

### Create executable

```bash
# Build for current platform
make build

# Build Linux only
make build-linux

# Build Windows only
make build-windows
```

Binary outputs:
- `bin/survival-bot-linux` (Linux)
- `bin/survival-bot.exe` (Windows)
